📊 数据API服务

供外部AI平台调用的MaxCompute数据查询接口

🚀 快速开始

本服务提供REST API接口，业务方AI平台可通过HTTP请求获取MaxCompute数据。

💡 提示：数分人员可在 管理后台 编写和管理SQL查询

调用示例（Python）

# 安装依赖 pip install requests # 调用接口 import requests response = requests.get('http://your-server.com/api/query/sales_report') data = response.json() # 处理返回数据 if data['success']: records = data['data'] # 数据列表 columns = data['columns'] # 字段名 row_count = data['row_count'] # 行数 print(f"共 {row_count} 行数据")

📖 API 参考

GET /api/queries

列出所有可用的数据接口

GET /api/queries

GET /api/query/{name}

执行指定SQL查询并返回数据

GET /api/query/{name}?limit=100
参数类型说明
limit整数最大返回行数（可选，默认1000）

POST /api/query

执行自定义SQL（临时测试用）

POST /api/query

请求体：

{ "sql": "SELECT * FROM table LIMIT 10", "limit": 100 }

响应格式

{ "success": true, "data": [ {"字段1": "值1", "字段2": "值2"}, {"字段1": "值3", "字段2": "值4"} ], "columns": ["字段1", "字段2"], "row_count": 100, "exec_time": 5.23, "query_time": "2026-07-03 16:30:00" }

错误处理

{ "success": false, "error": "错误信息" }
状态码说明
200成功
404查询不存在
500服务器错误
